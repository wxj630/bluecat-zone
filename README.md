运行后端服务器：

```shell position-relative overflow-auto
source ~/WorkSpace/SoftWare/miniconda3/bin/activate
python manage.py runserver
```


进入项目的 `frontend/` 目录，安装前端依赖项：

```shell position-relative overflow-auto
npm install
```

运行前端服务器：

```shell position-relative overflow-auto
npm run serve
```

至此前后端开发服务器就都启动了。

账号和原项目类似。
